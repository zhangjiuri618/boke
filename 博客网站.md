# 科技资讯博客网站开发方案（Vercel 部署）

## 项目目标

开发一个类似科技资讯门户的网站：

-   AI资讯
-   NAS教程
-   软件教程
-   工具导航
-   资源分享
-   文章博客
-   热门阅读排行

目标效果：

-   首页大 Banner
-   分类导航
-   卡片式文章布局
-   右侧热门文章
-   SEO 优化
-   支持后期 AI 功能扩展

------------------------------------------------------------------------

# 技术选型

## 前端

### Next.js 15

用途：

-   页面渲染
-   SEO
-   路由管理
-   服务端组件

### TypeScript

用途：

-   类型安全
-   提升代码维护性

### Tailwind CSS

用途：

-   快速开发 UI

### shadcn/ui

用途：

-   高质量组件库

### Framer Motion

用途：

-   页面动画效果

# Vercel部署流程

## 1. Github提交

``` bash
git add .
git commit -m "init project"
git push
```

## 2. Vercel部署

流程：

    Vercel

    ↓

    New Project

    ↓

    选择Github仓库

    ↓

    Deploy

自动完成：

-   安装依赖
-   构建
-   发布
